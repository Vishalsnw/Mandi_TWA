# Mandi TWA - Replit Project

## Overview
This is a Trusted Web Activity (TWA) application imported from GitHub. The original repository was empty except for a LICENSE file, so a basic PWA/TWA starter application has been created.

## Project Architecture
- **Frontend**: Vanilla HTML/CSS/JavaScript
- **Server**: Node.js HTTP server
- **Port**: 5000 (frontend)
- **Features**: 
  - Progressive Web App (PWA) capabilities
  - Service Worker for offline support
  - Web App Manifest for installability
  - Responsive mobile-first design

## Recent Changes
- 2024-11-23: Initial setup with basic TWA/PWA structure
  - Created HTML/CSS/JS frontend
  - Added Node.js server on port 5000
  - Implemented Service Worker
  - Added Web App Manifest
  - Configured for Replit environment

## Files Structure
- `index.html` - Main application page
- `style.css` - Application styles
- `app.js` - Client-side JavaScript
- `service-worker.js` - PWA service worker
- `manifest.json` - Web app manifest
- `server.js` - Node.js HTTP server
- `package.json` - Node.js dependencies

## Running the Application
The application runs on Node.js and serves static files. The workflow is configured to start the server on port 5000.
